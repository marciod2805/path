# Backend Service

This directory will house the Python backend for the Completionist Hub.

## Planned Stack
- **Framework**: FastAPI (High performance, easy async)
- **Database**: PostgreSQL
- **AI**: Integration with Google Gemini via Python SDK

## Setup (Future)
1.  Create virtual environment: `python -m venv venv`
2.  Install requirements: `pip install fastapi uvicorn`
3.  Run server: `uvicorn main:app --reload`
